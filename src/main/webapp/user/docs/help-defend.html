云防护

什么是云防护？

云防火墙是新一代防火墙技术，为网站提供一站式的安全解决方案，安全防护、智能调度，使网站在零部署、零维护情况下，防止诸如木马、零日攻击等各种网站安全问题发生，进一步提升网站访问速度，降低故障率，从而整体提升网站的用户体验。
· 加强安全性--完美的处理服务器的突发流量，有效的排查恶意用户非正常请求，吸收用户服务器的突发流量，防止木马攻击和网页内容篡改，加强系统的安全性，提高整体的服务质量。
· 全面可控--全面覆盖现有系统，安全防御，对应用层和网络层采用多元化的方法实现更加精细、准确、可控制的安全管理，有效解决应用层和网络层的安全问题。
· 智能稳定--专业技术防护主流的应用攻击，智能检测拦截，动态生成防御规则阻断攻击。为用户提供服务的同时，也提供了稳定性保障。为用户提供无缝式的访问体验。
· 完整性--并行处理技术，高效结构，包括传统的安全防御功能，提供完整的无死角防护服务。

创建云防护

· 用户登录控制台后，可在控制台内的云防护中，进行如下操作步骤：

1、点击管理页面中的“创建云防护”按钮：

2、在弹出的窗口中，可以输入创建云防护的名称后点击确定。

创建云防护规则

1、在你的云防护列表中点击需要设置的云防护名称，进入详细页面

2、点击创建规则按钮

3、在弹出的窗口中填入相关配置信息，也可在右侧选择我们为您提供的配置，点击确定

· 如需对添加好的规则进行修改，可在相应的规则后点击编辑进行修改

· 删除规则
