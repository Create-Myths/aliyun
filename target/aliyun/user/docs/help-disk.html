在弹出的创建硬盘对话框中，需填写您要创建的硬盘名称和大小：

硬盘操作

在硬盘管理界面，您可以看到列表列示的所有您账号下的硬盘及其详细信息，有硬盘名、容量、挂载的主机（如果未挂载则为空）、挂载的设备名、状态（分为可用和已挂载）、创建时间：

上方的操作栏则显示了您可以对硬盘进行的操作，有挂载、卸载、删除、克隆、扩容：

· 挂载

对于一个状态可用的硬盘，您可以将其挂载到任意的主机上。选中硬盘，点击挂载按钮，在弹出的对话框中，选择您要挂载的主机，确认即可完成挂载操作：

· 卸载

当您不需要使用挂载在某台主机上的硬盘时，可以随时对其进行卸载操作。选中您需要卸载的主机，点击卸载按钮，确认即可。需要注意的是，您在卸载之前，需要先从您的主机中umount磁盘，否则会导致数据丢失，且无法正常关机。

· 删除

如果您不需要某块硬盘了，还可以删除之。删除之前，需要保证硬盘没有挂载在主机上。一旦硬盘删除，其中数据也就不存在了，所以请事先做好备份工作。当您确保以上条件都达到时，选中硬盘，点击删除按钮，确认即可：

· 扩容硬盘

对于已有的硬盘，如果您不满意它的大小，也可以对其进行扩容操作，增加其容量。选择要扩容的硬盘，点击扩容硬盘按钮，在弹出的对话框中需填写您要扩容至多大，当然，不可以小于或等于原来的大小。

· 克隆硬盘

创建一块新的硬盘，除了直接创建外，还可以通过克隆已有硬盘的方式。通过克隆，可以完全复制一块硬盘，包括其上的数据。克隆操作也需要在硬盘未挂载在主机上的状态下进行，否则无法保证数据的实时性。
