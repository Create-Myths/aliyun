负载均衡

创建负载均衡

1、在负载均衡界面点击创建负载均衡按钮，如图：

2、在弹出的页面中，选择负载均衡前端IP的类型及带宽、计费方式等，页面右侧展示出配置和账户余额等信息，并根据选择统计费用

3、输入负载均衡的名称，选择负载均衡的协议类型，配置前端端口、主机监听端口，选择转发规则等，选择是否开启会话保持。

4、勾选需要加入到负载均衡组里面的主机

5、对负载均衡的各项参数确认后，点击”确认创建”,完成负载均衡申请

修改负载均衡配置

1、在负载均衡列表中，点击你要操作的负载均衡名称，进入该负载均衡规则管理界面。

点击“编辑”按钮对相应的规则进行编辑操作。

· 为负载均衡添加多端口规则

· 删除负载均衡多端口规则

· 登陆

点击“登录”，跳转至phpMyAdmin登录页面，然后输入用户名和密码，即可访问DB

删除负载均衡

勾选需要删除的负载均衡，点击”删除”，确认后删除成功。

FAQ

· 如果使用云内存、云硬盘等存储服务，是否还需要额外购买负载均衡？

不需要，游戏云的云内存采用分布式结构，自动进行负载均衡，无需额外购买负载均衡产品。
