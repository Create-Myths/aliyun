云数据库

什么是云数据库

云数据库架构在云端数据库集群上，通过云服务的方式让关系型数据库的可靠性更高，免去繁琐的维护工作，节约硬件成本，其具备以下特点：
性能出色——针对数据库高性能需求，采用高端高性能硬件配置，同时对数据库性能参数做了特定的优化。
管理方便——只需根据自己的数据空间、性能要求选择相应的UDB实例，数分钟后即可直接使用。
兼容性强——无缝兼容MySQL兼容的所有应用，不需要做任何改动，就可将应用程序迁移至UDB数据库上。
弹性扩展——可以非常方便地根据数据库的实际负载情况而进行实例的升级，从而获取更高数据库空间与性能。

创建DB实例

用户登录控制台后，可以在云数据库界面创建DB实例，具体步骤如下：

1、点击DB实例管理页面的创建DB实例按钮，如图：

2、在弹出的选择页面中，可以选择DB类型、实例类型和数量、计费方式，并根据您的选择统计费用，页面右侧会显示您的当前所选配置

DB类型——DB的类型以及版本，如mysql5.5、mysql5.1。请根据您对DB的类型和版本需求进行选择。
实例类型——DB的硬件配置，即内存和磁盘空间。请根据您对DB的硬件需求进行选择。
数量——填写您需要的DB的数量，默认申请一个，也可以选择多个进行批量创建。
根据您以上选择的资源，系统会统计显示您需要支付的费用。

3、 设置您要创建的实例的名称、端口号，以及用户名和密码以便于登录访问。端口号默认为3306，管理用户默认为root。为了您的数据安全，用户名和密码需要有一定的复杂度。

4、选择配置文件和备份策略

5、点击确定创建DB实例

FAQ

· 使用外部服务器可以访问游戏云的关系型数据库吗？

可以。关系型数据库默认访问地址是公网，可以通过外部服务器访问。如果配合云主机使用会大大的节省访问了流量。

· 如何访问UDB

web控制台访问：
选择需要访问的UDB实例，点击列表上方的“登录”，跳转至phpMyAdmin登录页面，然后输入用户名和密码，即可访问DB。
